# Operational Documents

This folder stores the unstructured business context for the analytics copilot.

## Why This Exists

Structured data tells us what happened.
These documents help explain:

- what a metric means
- how analysts should investigate anomalies
- when to escalate
- what operational patterns are considered likely root causes

## Current Document Groups

- `metric_definitions/`
- `runbooks/`
- `sops/`
- `policies/`
- `incident_notes/`

In a later step, these files will be chunked and indexed into a vector database for retrieval.
